# 🚌 Bus Ticket Booking System

## 🚀 Tech Stack
- Frontend: React.js, HTML, CSS, JavaScript  
- Backend: Java  
- Database: MySQL  

## ✨ Features
- User authentication (Login/Register)  
- Seat booking system  
- CRUD operations  
- Responsive UI
